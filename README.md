#Le programme suivant permet de réaliser dans l'ordre les actions suivantes:

	## 1- Récupération par SFTP du fichier dump de la base de données compréssée (DumpForFo.sql.7z)
	## 2- Décompréssion du fichier
	## 3- Démarrage du serveur du serveur Wamp
	## 4- Import du dump en base de données


#Les pré-requis à installer sont:

	## 1- winscp
	## 2- 7zip
	## 3- Wampserver
	## 4- Mysqlworbench ou HeidiSQL (accès aux données)

Les variables suivantes devront-être modifié.
